방법 1

  • Arduino IDE 실행
  • Library Manager 탭에서 Adafruit PWM Servo Driver Library 검색
  • 2.4.1 버전 다운로드

방법 2

  • https://github.com/adafruit/Adafruit-PWM-Servo-Driver-Library 사이트에서 레포지토리 zip파일로 저장
  • 상단 메뉴바에서 Sketch -> Include Library -> Add .Zip Library 선택
  • 다운로드받은 zip파일 선택

Arduino IDE 설정

공통

  • 상단 메뉴바 file → Perferences… → Additional Board Manager URLs. 란에 https://espressif.github.io/arduino-esp32/package_esp32_index.json 입력

ESP32CAM.ino 설정

  • 상단 메뉴바 Tools → Board → esp32 → AI Thinker ESP32-CAM 선택
  • ESP32-CAM 연결
  • 상단 메뉴바 Tools → Port 에서 연결된 기기 선택
  • 상단 메뉴바 Tools 에서 ESP32-CAM 세부설정 변경
    • CPU Frequency: “240MHz”
    • Core Debug Level: “None”
    • Erase All Flash Before Sketch Upload: “Disable”
    • Flash Frequency: “80MHz”
    • Flash Mode: “QIO”
    • Partition Scheme: “Huge APP (3MP No OTA/1MB SPIFFS)”
  • 업로드 전 Download 모드로 재부팅(IO0핀 접지 또는 호환보드 IO0버튼 누른 상태에서 리셋)
  • 컴파일 후 업로드

1_bucket_V1.0.ino 설정

  • 상단 메뉴바 Tools → Board → esp32 → ESP32 Dev Module 선택
  • ESP32 연결
  • 상단 메뉴바 Tools → Port 에서 연결된 기기 선택
  • 상단 메뉴바 Tools 에서 ESP32 세부설정 변경
    • CPU Frequency: “240MHz(WiFi/BT)”
    • Core Debug Level: “None”
    • Erase All Flash Before Sketch Upload: “Disable”
    • Events Run On: “Core 1”
    • Flash Frequency: “80MHz”
    • Flash Mode: “QIO”
    • Flash Size: “4MB (32Mb)”
    • JTAG Adapter: “Disabled”
    • Arduino Runs On “Core 1”
    • Partition Scheme: “Default 4MB with spiffs (1.2MB app/1.5MB SPIFFS)”
    • PSRAM: “Disabled”
    • Upload Speed: “921600”
  • 업로드 전 Download 모드로 재부팅(IO0핀 접지 또는 BOOT버튼 누른 상태에서 리셋)
  • 컴파일 후 업로드

ESP32 선 연결

ESP32 전원은 5V/2A 이상의 전원에서 안정적으로 동작함

PCA9685는 PWM, V+, GND로 이루어진 확장 핀 16세트가 0~15번까지 존재함. 4번 핀은 서보모터용으로 사용됨.

모든 전원부는 5V로 통일. PCA9685의 4번을 제외한 확장핀 V+와 GND핀을 활용해서 5V와 GND를 연결

배터리 잔량 측정 핀

5V - 저항 - 저항 - GND 순으로 직렬연결. 두 저항은 100KΩ 이상의 같은 값

두 저항 사이의 절반의 전압을 가진 노드를 ESP32 IO33핀에 연결

밥그릇 인식 핀

5V - CDS - 200KΩ - GND 순으로 직렬연결

CDS 소자와 저항 사이의 노드를 ESP32 IO25 핀에 연결

SG-90 서보모터 핀 연결

서보모터는 3가닥으로 이루어진 연결선이 제공됨.

주황색(PWM),빨강색(전원선),갈색(GND)로 이루어져 있음

주황색(PWM)PWM (4)
빨강색(전원선)V+ (4)
갈색(GND)GND (4)
