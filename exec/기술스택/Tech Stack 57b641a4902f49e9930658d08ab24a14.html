Kotlin - 1.8.0

  • 해당 기술 선택 이유

    선택이유

    1. Kotlin은 JVM(Java Virtual Machine) 위에서 작동하므로, Java와 함께 사용하는 것이 가능하고, 이미 존재하는 Java 코드베이스와 호환성을 가진다.

    1. Kotlin은 안드로이드 개발의 공식 언어이므로, 안드로이드 플랫폼의 모든 기능과 API에 대한 완전한 지원을 받는다.

🌠IoT

ESP32

  • 선택 이유와 설치 Script

    선택이유

    ESP32는 많은 장점이 존재합니다.

    • 저비용 : ESP32는 와이파이를 지원하는 칩셋 중에서 가장 저렴한 모듈 중 하나입니다.
    • 저전력 : 다른 마이크로컨트롤러에 비해 매우 적은 전력을 소비하며 전력을 절약하기 위해 딥 슬립 과 같은 저전력 모드 상태를 지원합니다 .
    • WiFi 지원 : ESP32는 쉽게 Wi-Fi 네트워크에 연결하여 인터넷에 연결하거나(스테이션 모드) 자체 Wi-Fi 무선 네트워크( 액세스 포인트 모드 )를 생성하여 다른 장치가 연결할 수 있습니다.
    • Arduino "프로그래밍 언어"와 호환 가능 : 공식 Arduino IDE 프로그램을 이용하여 Arduino 스타일로 ESP32를 프로그래밍할 수 있고, 대부분의 코드가 호환하여 동작합니다.

🌠AI

YOLOv5

  • 선택 이유와 설치 Script

    선택이유

    YOLOv5를 사용하여 고양이 객체 인식 인공지능을 구현할 때, 다음과 같은 장점이 있습니다:

    • 높은 정확도: 최신 딥러닝 기술을 활용하여 객체 검출 정확도를 높였습니다.
    • 실시간 처리: 실시간 객체 검출을 위해 최적화되어 있으며, 실시간 환경에서도 고양이를 실시간으로 인식하고 추적할 수 있습니다.
    • 다양한 크기의 객체 검출: 다양한 크기의 객체를 검출하는 데 강점이 있어, 고양이의 다양한 크기와 자세를 가진 이미지에서도 효과적으로 객체를 인식할 수 있습니다.
    • 경량화된 구조: 경량화된 구조로 GPU와 CPU에서 빠르고 효율적인 실행이 가능하며, 리소스 사용을 최소화하면서 고양이 객체 인식을 구현할 수 있습니다.
    • 다양한 플랫폼 지원: 다양한 플랫폼에서 실행할 수 있어, 모바일 장치나 임베디드 시스템 등 다양한 환경에서도 구현할 수 있습니다.

    따라서 YOLOv5는 고양이 객체 인식 인공지능을 구현하는 데 효과적인 선택입니다.

    설치 스크립트

    git clone https://github.com/ultralytics/yolov5  # clone
cd yolov5
pip install -r requirements.txt  # install

MobileNet v2

  • 선택 이유와 설치 Script

    선택이유

    1. 속도와 메모리 효율성이 좋다. MobileNet은 경량화된 구조로 알려져 있어서 일반적인 컴퓨터 비전 작업에 적합한 모델. 경량화된 모델 구조는 GPU 및 CPU에서 빠르게 실행되고 메모리 사용을 최소화하여 효율적인 클러스터링 분석이 가능하다.
    1. 전이 학습에 적합: MobileNet은 이미지 분류 작업에 대한 사전 훈련된 모델로 사용하기에 적합. 이전에 대규모 데이터셋 (예: ImageNet)에서 학습된 MobileNet 모델을 사용하여 고양이 사진들의 이미지 유사도를 분석하는데 활용할 수 있고, 작은 데이터셋에도 일반화 성능을 향상시키는 데 도움이 됩니다.
    1. 특징 추출: MobileNet은 이미지의 고수준 특징을 추출하는 데 강점을 가지고 있다. 특징 추출 능력을 기반으로 고양이 사진들의 이미지 유사도 분석에 유용할 수 있다.
    1. 확장성: MobileNet은 다양한 플랫폼에서 사용할 수 있는 모델로 적합.

    설치 스크립트

    tensorflow에 내장된 라이브러리 이기 때문에 tensorflow를 설치해서 사용하면 됨. tensorflow > keras > mobilenet_v2

    pip install tensorflow

# import해서 사용법
from tensorflow.keras.preprocessing import image
from keras.applications.mobilenet_v2 import MobileNetV2

Tensorflow 2.12.0

  • 선택 이유와 설치 Script

    선택이유

    Tensorflow는 구글에서 개발한 오픈소스 기계 학습 프레임워크입니다. 즉, 기계 학습 모델을 만들고 학습시키기 위한 도구이며, 딥러닝 모델을 구현하기 위한 도구 중 하나입니다.

    Tensorflow를 사용하는 이유

    1. 빠른 학습과 데이터 처리: Tensorflow는 병렬 연산을 지원하고, GPU를 활용하여 대규모 데이터를 빠르게 처리할 수 있습니다.
    1. 다양한 모델과 알고리즘: Tensorflow는 다양한 모델과 알고리즘을 지원합니다. 따라서, 다양한 영역에서 사용 가능합니다.
    1. 확장성: Tensorflow는 분산 처리가 가능합니다. 따라서, 여러 대의 서버에서 모델 학습을 수행할 수 있습니다.
    1. 커뮤니티: Tensorflow는 구글에서 개발하고 있으며, 많은 개발자들이 사용하고 있습니다. 따라서, 커뮤니티에서 다양한 라이브러리와 모델을 공유하고 있습니다.

    Tensorflow를 사용하여 기계 학습 모델을 구현하고 학습시키면, 다양한 분야에서 사용 가능한 예측 모델을 만들 수 있습니다.

    설치 스크립트

    # Requires the latest pip
pip install --upgrade pip

# Current stable release for CPU and GPU
pip install tensorflow
